Weekday Calculator

Enter a number between 1 and 7:

In this example, we have an HTML input element with an id attribute of "numInput" and a HTML button element with an onclick attribute that calls the calculateWeekday function when clicked. The calculateWeekday function gets the value of the input field using the getElementById function and stores it in the num variable. It uses the num variable to display the corresponding weekday using the alert function. If the entered value is not between 1 and 7, it asks you to enter the number between 1 to 7.
